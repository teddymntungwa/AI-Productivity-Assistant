# AI Workplace Productivity Assistant

## Project Overview

AI Workplace Productivity Assistant is a modern SaaS-style web application designed to help professionals automate common workplace tasks using Artificial Intelligence.

The application provides a centralized productivity dashboard featuring multiple AI-powered tools for email creation, meeting note summarization, task planning, research assistance, and conversational AI support.

Built with React, Next.js, and Tailwind CSS, the platform delivers a responsive and professional user experience across desktop, tablet, and mobile devices.

---

## Features

### Dashboard

* Responsive SaaS-style layout
* Left sidebar navigation
* Top header
* Welcome banner
* Quick action cards
* Recent activity section
* Responsible AI disclaimer
* Mobile-friendly design

### Smart Email Generator

Generate professional emails using:

* Recipient
* Role
* Tone
* Purpose
* Key points
* Desired outcome

Includes:

* Generate button
* Editable output
* Copy functionality
* Regenerate action

### Meeting Notes Summarizer

Convert meeting notes into structured summaries.

Output includes:

* Summary
* Action Items
* Decisions
* Blockers
* Next Steps

### AI Task Planner

Create organized project plans from goals and workload descriptions.

Output includes:

* Prioritized tasks
* Deadlines
* Dependencies
* Recommended execution order

### AI Research Assistant

Generate research insights from a topic and objective.

Output includes:

* Summary
* Insights
* Recommendations
* Further questions

### AI Chatbot Interface

Interactive AI assistant featuring:

* Chat history
* Prompt suggestions
* Message input
* Send button
* Mock AI responses

### Responsible AI Notice

The application displays the following disclaimer:

> AI-generated content may be inaccurate or incomplete. Please review, edit, and verify important information before use.

## Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/ai-workplace-productivity-assistant.git
cd ai-workplace-productivity-assistant
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start the Development Server

```bash
npm run dev
```

### 4. Open the Application

Visit:

```text
http://localhost:3000
```

### 5. Build for Production

```bash
npm run build
```

### 6. Run Production Build

```bash
npm start
```
